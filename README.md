## 简介

基于 Vue3、Vite、TS、Naive UI、Pinia 的项目，使用了最新的技术栈，代码规范已设置 Eslint+Prettier、动态权限路由

## 特性

- :使用 Vue3/Vite2 等前端前沿技术开发，使用 pnpm 作为包管理器
- :使用 Ts 开发
- : 基于 windicss
- : 代码规范基于 Eslint+Prettier
- : 封装请求
- ：UI 使用 Naive UI
- : Sass 使用

## 方便开发

引入 unplugin-icons

## 接口封装

基于 axios 以及 nprogress 配置
